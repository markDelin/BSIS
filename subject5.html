Flowchart - represent the process.

Basic symbol of flowcharts

  • Oval (Terminator) -Indicates the start or end of a process.
  • Rectangle (Process) - Represents a process, task, or action to be performed.
  • Diamond (Decision) - Represents a decision or branching point where the flow can take different paths based on a condition.
  • Parallelogram (Input/Output) - Represents input (e.g., user input) or output (e.g., displaying results).
  • Arrow (Flow Line) Shows the direction of the flow from one step to another.

    • Fundamental Data Types

    1. int (Integer) - 2/4 bytes

    2. float (Floating-point) - 4 bytes

    3. double (Double floating-point) - 8 bytes

    4. char(Character) - 1 byte

    5. wchar_t (Wide character) - 2 bytes

    6. bool (Boolean) - Can store true or false

    7. void - Represents an empty type

    C++ Data Types

    1. int - Can hold whole numbers.

    2. double - Can hold numbers with decimals or in exponential notation.

      Ex. 45E10 (means 45 × 10^10)
        2E2 (means 2 × 10^2)

    3. char - Used to store single characters.

      Ex. char name = 'c';

      Ex. for a string: char name[10] = "Ericson";

    4. wchar_t - Used for wide characters.

      Ex. wchar_t text = L'C';

    5. bool - Used to store boolean values (true or false).

      Ex. bool cond = false;

    6. void - Represents an empty value or no data.

    Modifiers

    signed - Can hold positive and negative values.

    unsigned - Can only hold positive values.

    short - 2 bytes (range: -32,768 to 32,767)

    unsigned short - 2 bytes (range: 0 to 65,535)

    long - Used for larger integers.

    unsigned long - Positive large integers.

    long long - Very large integers.

    unsigned long long - Positive very large integers.

    long double - Used for large floating-point numbers (12 bytes).

    signed char - Range from -128 to 127.

    unsigned char - Range from 0 to 255.

    Constants

    Constants are values that do not change.

      Ex. const double PI = 3.14;

      const int MAGIC_NUMBER = 42;

      Note: Constants should be capitalized by naming convention.

    Operators

    1. Addition (+) - Adds two numbers.

  • Ex. 5 + 3 results in 8.

    • 2. Subtraction (-) - Subtracts one number from another.

  • Ex. 7 - 2 results in 5.

    • 3. Multiplication (*) - Multiplies two numbers.

  • Ex. 6 * 4 results in 24.

    • 4. Division (/) - Divides one number by another.

  • Ex. 8 / 2 results in 4.

    • 5. Modulo (%) - Used for finding the remainder when dividing two numbers.

  • Ex. 10 % 3 results in 1 (since 3 goes into 10 three times, leaving a remainder of 1).
